This application takes in a rubix cube state and outputs the directions to solve it.

One can click on the interactive cube UI to make it match the state of a rubix cube.

One can also click the "Set Example" button to automatically generate an example cube.

Once the cube is inputed, press solve. The directions to solve the inputed state of the cube will appear on the right of the cube.

The directions can be clicked and will display the state of the cube after each direction is executed. This can help one solve the cube, as they can follow along and make sure their cube matches the UI cube.

This is a link to the instruction guide used to write this solving algorithm. Navigate to the page in the pdf where the directions are listed. These are the same directions which the algorithm outputs.

Additionally...

Y Rot: Rotate the cube about the y-axis in the right direction

Y Rot': Rotate the cube about the y-axis in the left direction

X Rot: Rotate the cube about the x-axis in the up direction

X Rot': Rotate the cube about the x-axis in the down direction
